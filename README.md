# lockkliye

An iCloud-notes alternative for secure and organized storing of custom markdown-like notes.

## Features

- Ability to tap-to-edit single words, lines or paragraphs
- Add customized seperators
- Manual and automatic backup of files in a variety of different file-formats
- iCloud like tree-navigation and folder managment
